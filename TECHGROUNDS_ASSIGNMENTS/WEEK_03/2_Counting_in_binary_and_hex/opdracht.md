 Exercise:

| **Decimal** | **Binary** |
| ----------- | ---------- |
| 16          |            |
| 128         |            |
| 228         |            |
| 112         |            |
| 73          |            |

- Translate the following binary numbers into decimal:

| **Binary** | **Decimal** |
| ---------- | ----------- |
| 1010 1010  |             |
| 1111 0000  |             |
| 1101 1011  |             |
| 1010 0000  |             |
| 0011 1010  |             |

- Translate the following decimal numbers into hexadecimal:

| Decimal | Hex |
| ------- | --- |
| 15      |     |
| 37      |     |
